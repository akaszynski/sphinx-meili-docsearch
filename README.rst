sphinx-meili-docsearch
======================

.. image:: https://img.shields.io/pypi/v/sphinx-meili-docsearch.svg
   :target: https://pypi.org/project/sphinx-meili-docsearch/
   :alt: Package on PyPI

To use it, add ``sphinx.meili.docsearch`` as a Sphinx extension:

.. code:: python

   # conf.py

   extensions = [
       "sphinx.meili.docsearch",
   ]
   ...
